\section*{Data Availability}\label{data-availability}
\addcontentsline{toc}{section}{Data Availability}

Field-collected data, \texttt{neonSoilFlux} calculated outputs, and
manuscript-generating code are archived via Zenodo:
\url{https://doi.org/10.5281/zenodo.17516319} (Zobitz \& Zimmerman,
2025). All analyses in this manuscript were based on the version of
\texttt{neonSoilFlux} tagged v2.9.0 on the package's GitHub repository
\url{https://github.com/jmzobitz/neonSoilFlux}. This version of the
package is also archived at Zenodo under the following DOI:
\url{https://doi.org/10.5281/zenodo.17624494} (Zobitz et al., 2025).
